

## 1. Clone the project

```sh
git clone git@github.com:nemanja85/Library-Form-Submission.git
```
## 2. Navigate to the repository

```sh
cd library-form-submission
```

## 3. Install the required dependecies

```sh
npm install
```

# Running the Project
### To start the project, use the following command:

```sh
npm run dev
```
